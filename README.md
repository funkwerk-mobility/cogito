# Installing and usage

Extract dmd.2.098.1.linux.tar.xz into tools, so you have the tools/dmd2
directory.

Run `./make.rb release`.
Run `./build/cogito sample/sample.d`.

# Example output

```
sample/sample.d:
  f:
    Location (line): 3
    Score: 4
```
